# 🔴 Connect Four

This is a simple **Connect Four** game built with React.

It was originally bootstrapped with [Create React App](https://github.com/facebook/create-react-app) in 2020 and later migrated to Vite.

This project is kept as an archive and is no longer actively developed.

## 📌 Status

⚠️ **Archived / Completed**

This project is finished and will not receive further updates.  

## 🎮 About the project

This is a classic **Connect Four** implementation:

- Two-player turn-based gameplay
- Win detection (horizontal, vertical, diagonal)
- Simple UI built with React
- Fully playable in the browser

## 🛠️ Tech Stack

- React
- Vite (migrated from Create React App)
- JavaScript / JSX
- CSS


## 🚀 Running the project

To run locally:

```bash
npm install
npm run dev
```
